# Interactive Memory Matching Game (React.js)

A responsive, single-player card matching game built with React, focusing on component-based architecture and efficient state management.

#Technology Stack

| Component | Technology | Role |
| Frontend Framework | React.js | Handling UI updates, component lifecycle, and state management. |
| Language | JavaScript (ES6+), ReactJS | Implementing game logic and component functions. |
| Styling | CSS3 | Component styling and responsiveness. |
| Tools | npm/Yarn, Git/GitHub | Package management and version control. |

# Installation and Local Run

To get this project running on your local machine, follow these steps:

#Prerequisites

* Node.js and npm (or Yarn) installed

#Setup Steps

1.  Clone the Repository:
    ```bash
    git clone [https://github.com/Vishh1610/memory-game.git](https://github.com/Vishh1610/memory-game.git)
    cd memory-game/memory-card  # Adjust path if 'memory-card' is the project folder
    ```

2.  Install Dependencies:
    ```bash
    npm install
    # OR
    yarn install
    ```

3.  Start the Application:
    ```bash
    npm start
    # OR
    yarn start
    ```
